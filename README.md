# Phishing Email Detection System

This repository contains the implementation of a machine learning–based system for phishing email detection, developed as part of a Bachelor’s thesis.

## Project Overview
The system performs automatic phishing detection using text and URL-based features extracted from emails. It includes model training, evaluation, and deployment through a REST API, as well as real-time inbox monitoring and visualization.

## Key Features
- Text and URL feature extraction
- Logistic Regression / SGD-based classification
- REST API for email analysis
- IMAP inbox monitoring
- Visualization and reporting of results

## Technologies
- Python
- scikit-learn
- FastAPI
- Docker
- Grafana & Loki

## Repository Structure
- `src/` – source code
- `models/reports/` – evaluation plots and reports
- `monitoring/` – monitoring configuration
